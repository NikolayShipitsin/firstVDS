чтобы запустить API и воркеры обработки файла 
нужно выполнить docker-comose up --build
API будет доступно по адресу http://localhost:8005/
краткая документация доступна по адресу http://localhost:8005/docs
Также для демонстрации работы  endpoint в репозитории есть коллекция postman
Т.к в задании указано, что файл может содержать много строк, то файл обрабатывается построчно(через генератор)
Для горизонтального масштабирования можно запустить несколько контейнеров с сервисом worker,
например docker-compose up --scale worker=5

Функционал для работы с хранилищем s3 не реализован, т.к не хватило времени на изучение документации по работе с s3
(в документации пока не нашел как обрабатывать файлы из s3 построчно
и как при этом обрабатывать ошибки, возникающие из-за нестабильности соединения)
